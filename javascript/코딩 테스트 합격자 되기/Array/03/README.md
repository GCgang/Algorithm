## 문제 (03.js)

[03 두개 뽑아서 더하기](https://school.programmers.co.kr/learn/courses/30/lessons/68644)

권장 시간 복잡도: `O(N^2 log(N^2))`

## 풀이

1. 배열에서 두 수를 선택하는 선택하는 경우의 수 구하기
2. 과정 1에서 구한 수를 새로운 배열에 저장
3. 배열에서 중복된 값을 제거하고, 오름차순으로 정렬 후 반환
